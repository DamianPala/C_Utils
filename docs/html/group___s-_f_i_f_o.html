Detailed Description

This module contains implementation of Static FIFO.

S-FIFO module contains implementation of Static FIFO queue. Static means that memory is static allocated.

Items in this FIFO have to be fixed size and also size of queue have to be fixed. Items size is in bytes, FIFO size is in items number. Buffer intended for use by queue is automatically allocated. Queue can operate in mode when pushing operation overwrite last item with new one. This mode can be set via SFIFO_OverwriteLastItems function.
